## Implementation Strategy

### Notification Triggers:
- Users receive instant notification when rewards are earned
- Weekly summary of available rewards
- Push notifications for milestone progress (e.g., "3 more deals until your next reward!")

### Gamification:
- Progress bars for milestone rewards
- Badge showcase on profiles
- Leaderboard for top earners
- Streak trackers for daily/monthly activity

### Anti-Gaming Measures:
- Review verification for perfect rating rewards
- Time-based validation for login streaks
- Real deal completion verification
- Duplicate prevention for referrals
- **Stripe verification required for verified referral rewards** (prevents bot/spam accounts)

## Verification Process for Verified Referrals

### Stripe Identity Verification:
1. **User signs up via referral link** ‚Üí Invite marked as "accepted"
2. **User completes Stripe verification:**
   - Government-issued ID verification
   - Liveness check (selfie)
   - Address verification
3. **Verification approved** ‚Üí Invite marked as "verified"
4. **Agent reward triggered** ‚Üí 300 IMPACT at 20 verified invites, 1500 IMPACT at 100

### Benefits:
- Prevents bot/spam accounts
- Ensures quality user acquisition
- Builds trust in the platform
- Prepares users for payments/withdrawals

### Important Notes:
- Regular referral rewards (30 IMPACT per signup) are awarded immediately
- Verified referral bonuses require Stripe verification completion
- Users can complete verification anytime after signup
- Retroactive counting: If user verifies later, it counts toward milestone
